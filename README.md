# claude-pluginz

Marketplace catalog called `claude-pluginz` that lists my plugins.

## Installation

```shell
/plugin marketplace add sashadev-sky/claude-pluginz
/plugin install session-logger@claude-pluginz
```

## [Commands](./commands)

1) **`create-command`**:
   - Ex. `/claude-commandz:create-command analyze-deps "Analyze dependencies for outdated packages"`
